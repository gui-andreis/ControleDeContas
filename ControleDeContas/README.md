# 🏦 Controle de Contas - Console App em C#

Este projeto é um sistema simples de controle de contas desenvolvido em C# para praticar conceitos fundamentais de programação e organização de código.

## 🚀 Funcionalidades

- Cadastro de contas
- Remoção de contas com validação
- Listagem de contas
- Busca por nome (LINQ)
- Ordenação por nome e saldo
- Agrupamento por região
- Cálculo do total de saldo
- Persistência de dados em arquivo JSON

## 🧠 Conceitos Praticados

- Programação Orientada a Objetos (OOP)
- Encapsulamento
- Separação de responsabilidades (Services e Queries)
- Uso de LINQ
- Manipulação de arquivos (FileStream)
- Serialização e Desserialização JSON
- Uso de IReadOnlyList
- Organização de projeto em pastas
- Versionamento com Git

## 📂 Estrutura do Projeto

- Models → Classes de domínio
- Services → Lógica principal do sistema
- Queries → Operações com LINQ
- Data → Arquivo JSON de persistência
- Program.cs → Pipeline de execução

## 🎯 Objetivo

Projeto criado com foco em aprendizado e prática de fundamentos de backend em C#.
